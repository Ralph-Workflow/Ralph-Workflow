# Activity-Theater Guard

## What this is

A permanent guard against the mistake I made in May–June 2026: running a complex artifact pipeline for 3+ weeks (107 scripts, 50 blog posts, 94 Telegraph articles) with **zero real-world movement** — Codeberg stars flat at 12 the whole time — and never stopping to escalate.

## The rule (also in AGENTS.md)

If I'm operating a system that produces artifacts (scripts, content, posts, analyses) for a specific goal, I must check **real-world outcome metrics** at least every **7 days**. If 7 consecutive days pass with zero measurable external impact toward the stated goal:

1. **STOP** all artifact production immediately
2. **ESCALATE** to the human with the exact blocker and what they need to do
3. **DO NOT** continue optimizing internal machinery

## What "theater" looks like

| Looks like work | Actually is |
|---|---|
| Script A produces analysis | Analysis feeds into script B |
| Script B produces report | Report feeds into verifier C |
| Verifier C finds issues | Issues fed into fixer D |
| Fixer D generates patches | Patches applied back to script A |
| **External needle: flat** | **→ activity theater** |

If the output chain loops back into itself without reaching a real person, it's theater. Stop.

## What "real-world impact" means

Any signal from outside the system:
- Codeberg stars / watchers / forks going up
- A human comment, issue, or PR
- A measurable increase in downloads or visits
- A social share or mention by someone not me
- An email or contact form submission

Internal artifacts (number of scripts written, blog posts published, Telegraph articles cross-posted, audits completed) are **not** real-world impact.

## Per-action test (stronger than the weekly guard)

Before every action: **"Is what I'm about to do driving marketing results?"**

| If the answer is… | Then… |
|---|---|
| A clear yes | Do it |
| "Sort of" or "indirectly" | Stop. Find the direct path or skip it. |
| "No, but I'm optimizing/cleaning/fixing" | Stop. Optimization that doesn't reach people isn't marketing. |
| "No" | Obviously stop. |

Writing scripts, fixing dashboards, refining internal systems, running audits — none of that is marketing. Reaching people, moving them through a conversion path, getting them to try the product — that's marketing. If you can't show how the action leads to a human doing something, don't do it.

## Activation

This guard activates when I start or sustain any artifact-production system. It's not optional. If I'm in a session and find myself generating artifacts without checking external metrics within the last 7 days, the guard triggers and I must halt.
