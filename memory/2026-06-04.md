
## SECURITY INCIDENT: GitHub PAT leaked in commit history (2026-06-04 ~02:00 CEST)

**Email received:** GitHub detected a Personal Access Token in commit content and auto-revoked it.

### Root Cause
- **Commits on May 10-12, 2026** included a hardcoded GitHub PAT (`github_pat_11AEY3SZI0...`) in `agents/unblocker/run.py` (full token) and `memory/2026-05-09.md` (partially redacted).
- **3 commits contained the token:**
  - `70aa742` (May 10) — memory file with partial redaction
  - `5a2babe` (May 12) — run.py with full plaintext token written in code
  - `6fb633a` (May 10) — run.py with full plaintext token in rename
- **Sync cron** (sync_to_github.sh, deployed May 20 via e0801ee) pushed all prior commits to GitHub mirror, exposing the token in GitHub's history.
- **GitHub's secret scanning** detected it and revoked the token.
- No pre-commit secret scanner existed at the time.

### Damage Assessment
- Token was already read-only (noted in the May 9 memory file as "cannot file issues/PRs; all write operations return 403")
- GitHub auto-revoked it, so it can't be used
- No indications of unauthorized access (token was write-limited)
- File on current HEAD is clean — the token was removed from run.py in a later commit

### Immediate Actions Taken
1. ✅ Removed partial redacted token from `memory/2026-05-09.md`
2. ✅ Created `.githooks/pre-commit-secret-scan` — catches 12+ token patterns
3. ✅ Installed on workspace repo (via git config core.hooksPath)
4. ✅ Installed on Ralph-Site repo (via git config core.hooksPath)
5. ✅ Hardened `.gitignore` — added blanket `*_token*` and `*.token` patterns
6. ✅ Committed prevention system (commit 2470979)
7. ✅ Added zero-tolerance security rule to AGENTS.md

### Prevention
- **Pre-commit hook** blocks any commit containing GitHub PATs, OAuth tokens, OpenAI/Anthropic/API keys, AWS keys, Slack tokens, or PyPI tokens
- **AGENTS.md Red Line**: "Never hardcode credentials, API keys, tokens, or secrets in committed files"
- **`.gitignore` updated** with broader secret file patterns
- All repos have hooks installed
