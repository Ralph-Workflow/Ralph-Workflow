# Start Here

Pick one real task you already want done, run it unattended, then judge
the result by what the software actually does and what checks ran.

## Prerequisite

One supported agent CLI installed and authenticated on your machine.
The selection and trust-boundary story is in
[`agents`](ralph-workflow/docs/sphinx/agents.md).

## First run

The canonical six-step sequence (install → init → diagnose → spec →
run → review) lives in the
[`README.md`](README.md#first-run) first-run block. The pre-flight
check is run by the same six-step sequence; the long-form walkthrough
is in [`Getting started`](ralph-workflow/docs/sphinx/getting-started.md).

## Success looks like

After `ralph` returns, the finish-receipt artifact names the change,
the checks, and the review focus. Run the program against your real
environment, exercise the feature, then decide the next action.
