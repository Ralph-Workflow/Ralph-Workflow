SUBMISSION TEXT (Lobsters — "Show" style, more direct):

Ralph Workflow is an open-source AI coding orchestrator built around a simple idea: structure the session as a loop, not a prompt chain.

- Planning loop: turns a task into a runnable spec
- Development loop: executes against the spec, iterates, runs checks
- Review loop: surfaces a clean diff you'd actually merge

Works with Claude Code, Codex CLI, or OpenCode. Free and open source. No hosted component required.

The default workflow works as-is for a real task today, or you build your own workflow on top.
