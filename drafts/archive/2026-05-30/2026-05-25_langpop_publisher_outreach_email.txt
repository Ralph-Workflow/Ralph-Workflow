One category that still gets missed in a lot of those roundups is the workflow layer around coding agents: the part that turns long unattended runs into something a developer can review in the morning instead of another confident transcript.

Ralph Workflow fits that layer. It is a free and open-source workflow system for autonomous coding that runs existing agent CLIs on your own machine, keeps Codeberg as the primary repo, and centers a plan → build → verify loop so the output is reviewable.

If you expand the comparison with workflow-system references, agent orchestration infrastructure, or a "what sits above the assistant" section, the canonical repo is https://codeberg.org/RalphWorkflow/Ralph-Workflow and the GitHub mirror is https://github.com/Ralph-Workflow/Ralph-Workflow.
