# HN Post: "I let AI code for 4 hours unattended"
**Title:** Show HN — I built a tool that runs AI coding agents for 4+ hours while you sleep (spec-driven, reviewable commits)

**Body:**

I built Ralph Workflow (github.com/ghuntley/ralph) — a CLI orchestrator for AI coding agents that runs unattended for hours.

The premise: you write a spec, run `ralph`, and it:
1. Plans the implementation
2. Develops the code  
3. Verifies against the spec
4. Loops until done
5. Commits incremental progress

No babysitting. No API key juggling. No surprise costs.

Key difference from other AI coding tools: it's spec-driven. The AI can't deviate from your SPEC.md. Every commit is reviewable before the next iteration starts.

The catch: it's a CLI tool. No GUI. You need to know what you want to build.

Working on: team analytics dashboard, shared run history, one-click deploys.

Feedback welcome — especially from devs who've tried similar tools.
