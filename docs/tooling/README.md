# Tooling Notes

This directory collects notes about the tooling that ships with and
supports the maintained Python runtime of Ralph Workflow.

## Who this is for

Contributors and operators who want to know what tooling the Python
package depends on, and where to look when something in the toolchain
needs to be replaced or upgraded.

## Read this first

- **[python-tooling.md](python-tooling.md)** — the Python toolchain
  (uv, ruff, mypy, pytest, hatch) and the versions Ralph Workflow
  pins. This is the source of truth for "what tool runs which
  command" across the codebase.
