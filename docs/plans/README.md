# Implementation Plans

This directory holds implementation plans for the **maintained Python
runtime** of Ralph Workflow.

## Who this is for

Contributors and reviewers who want the design history and the agreed
shape of a Python-era change before they read the code that landed.

## Read this first

Plans are kept recent-first in their filename. The most recent plan is
the one to read first if you want the current shape of the Python
package.

- The full Python-era plan index is the directory listing below.
- Each plan describes a single change with: scope, file targets,
  acceptance criteria, work units (when parallelizable), and the
  expected verification command.

## Maintained: Python-era plans

The plans in this directory all describe the current Python package
under `ralph-workflow/`. They are NOT the Rust-era plans — those are
archived under [`../legacy-rust/plans/`](../legacy-rust/plans/README.md).
