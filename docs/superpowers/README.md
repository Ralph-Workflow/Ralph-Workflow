# Superpowers Docs

This directory collects notes about the Superpowers tooling that ships
with Ralph Workflow.

## Who this is for

Contributors and operators who want background on the Superpowers
skill bundles, spec layouts, and plan metadata used by Ralph Workflow.

## Subdirectories

- [`plans/`](plans/) — Superpowers plan metadata for the maintained
  Python package.
- [`specs/`](specs/) — Superpowers spec metadata for the maintained
  Python package.
