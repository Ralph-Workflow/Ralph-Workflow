# MCP Docs (package)

This directory collects MCP-specific reference and cookbook material
for the **maintained Python package** at `ralph-workflow/`.

## Who this is for

Contributors and operators who want to debug an MCP-specific issue or
extend the MCP surface of the Python package.

## Read this first

- **[mcp-servers.md](mcp-servers.md)** — reference for the standalone
  `ralph-mcp` server and how it relates to the rest of the package.
- **[web-search.md](web-search.md)** — the web-search MCP helper
  (transport, retry, and rate-limit notes).
- **[web-visit.md](web-visit.md)** — the web-visit MCP helper
  (transport, redirect policy, and content-type notes).
