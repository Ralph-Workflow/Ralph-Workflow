> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a second Ralph Workflow-managed namespaced design skill bundle sourced through Open Design discovery while fetching the actual upstream skill content rather than shipping Open Design pointer files.

**Architecture:** Keep the existing 17 workflow skills unchanged, add an Open Design-derived namespaced bundle in the shared shipped manifest, and teach the Node sync pipeline to resolve catalog entries to true upstream content before materialization into the flat runtime directory.

**Tech Stack:** Python 3.12, pytest, Node.js sync helper, JSON manifest/metadata, existing Ralph skills runtime/install pipeline.
